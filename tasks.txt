Page requirements:
	- responsive
		- displays well on mobile devices (tablets/phones)
		- use Bootstrap 3/Foundation 4 for existing content
	- aesthetically appealing
		- flat, geometric buttons, tile-like
		- use vector graphics (look good on HD displays)
		- use console type for headers (Consolas?) and sans-serif for other text
		- look into CS related images/backgrounds
		- minimal white space
		- look into a one-page layout, used more often these days <http://onepagelove.com/>
	- code guidelines
		- when using CSS templates, remove any unnecessary code
		- comment everything for maintainability
		- look into nanoc/ruby for development
